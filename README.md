
# group-order

Click on food items from a few resturants to build up a group order list. Built for use at Enspiral Dev Academy.

## how

### install

```
git clone https://github.com/troy-lamerton/group-order
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
